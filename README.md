--PENDU--

PENDU est un jeu gratuit créé en C++11 pour la console Windows.

C'EST QUOI, UN PENDU?
Voici la page Wikipédia dédié au jeu: https://fr.wikipedia.org/wiki/Le_Pendu_(jeu)

Le jeu du Pendu consiste à deviner un mot en proposant des lettres, et si ces lettres ne figurent pas dans le mot,
un dessin de pendu se dessine petit à petit.

COMMENT JOUER?
Executez le jeu.
Vous vous trouverez devant le titre du jeu, appuyez sur ESPACE.
Ensuite choisissez entre ces 2 modes de jeux:

-Player vs CPU: Vous jouez contre votre processeur, il choisit un mot au hasard dans un fichier.
-Player vs Player: Vous jouez contre un être humain, il devra choisir un mot que vous devrez deviner.

Pour confirmer, appuyez sur 'Y'.

Vous pourrez ensuite jouer au jeu :)
